==================================================================================
TROUBLESHOOTING
==================================================================================

"npm run dev won't start"
→ Check .env.local is configured
→ Verify Supabase credentials

"Can't submit booking"
→ Check database tables exist in Supabase
→ Check browser console (F12) for errors
→ Verify database connection

"Build errors"
→ Delete .next folder
→ Run: npm install
→ Try: npm run build again

More help in the documentation files!
